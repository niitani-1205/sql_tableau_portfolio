# SQL × Tableau BI ポートフォリオ

本リポジトリは、**Kaggleの架空顧客データ**を使用して、  
顧客の離脱率（Churn Rate）を SQL で分析し、Tableau で可視化したポートフォリオです。  
データ抽出から可視化までの一連のBI分析プロセスを再現しています。

---

## 📂 フォルダ構成

| フォルダ名 | 内容 |
|-------------|------|
| `Source_Data` | 分析に使用したKaggleの架空データ |
| `SQL_Queries` | MySQLで実行したクエリ（6つ） |
| `CSV_Results` | SQLの実行結果（CSV形式5つ)|
| `PNG_tableau` | Tableauで作成した可視化グラフ（PNG形式3つ） |

---

## 🧮 分析テーマ（概要）

SQLで顧客属性や契約内容別の離脱率を算出し、  
Tableauでグラフに可視化しました。 

【主な分析内容】
- 契約期間や料金帯による離脱率の違い  
- 性別・契約形態・請求方法による傾向分析  
- 契約タイプ別の離脱状況の比較  

---

## 🧰 使用ツール・環境

- **SQL**：MySQL 8.0  
- **BIツール**：Tableau Public  
- **可視化出力形式**：PNG  
- **OS**：Windows 11
